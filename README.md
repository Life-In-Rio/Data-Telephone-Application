# 📱 Database Telephone App

Aplikasi database telephone sederhana berbasis Flutter yang dibuat untuk menyimpan dan mengelola data kontak secara praktis.  
Data disimpan secara lokal menggunakan SQLite sehingga aplikasi dapat digunakan tanpa koneksi internet.

---

## ✨ Fitur Utama

- ➕ Tambah Data Kontak
- ✏️ Edit Data Kontak
- 🗑️ Hapus Data Kontak
- 💾 Penyimpanan Data Lokal menggunakan SQLite
- 📱 Tampilan UI sederhana dan mudah digunakan

---

## 🛠️ Teknologi yang Digunakan

- Flutter
- Dart
- sqflite
- path_provider
